# 功能：Project 表单视觉刷新

- 状态：已验收
- 负责人：shaoqian.li
- 创建日期：2026-07-13
- 验收日期：2026-07-13

## 背景

Project 创建/编辑弹窗在深色全局主题下使用默认 Element Plus 输入框，截图中表现为大面积
深色弹窗配高亮白色输入框，字段间距过大，层次不清晰；同时截图仍出现旧版云厂商和默认
集群字段，说明运行页面可能没有加载最新 ProjectCenter 代码。

## 当前行为

- 全局 `.el-dialog` 使用深色背景。
- Project 表单字段较多，默认输入框在深色弹窗里对比过强。
- 表单分区存在，但视觉上仍像一张长表。

## 目标行为

Project 创建/编辑弹窗应呈现为清晰、克制的工作台表单：浅色表单面板、紧凑输入控件、
明确分区、固定底部操作区，并保留现有提交校验和防重复提交逻辑。

## 范围

- 包含：优化 `ProjectCenter.vue` 弹窗的局部样式。
- 包含：降低输入框高度和圆角，减少刺眼白块。
- 包含：增强分区卡片、提交说明和 footer 的视觉层次。

## 非目标

- 不包含：改后端字段、API 或 Project 数据模型。
- 不包含：抽象全局表单设计系统。
- 不包含：修改成员、集群、Registry 子资源表单。

## 验收条件

- [x] Project 弹窗不再呈现深色大背景配巨大白色输入框的观感。
- [x] 表单分区清晰，输入框高度、圆角和间距适合工作台场景。
- [x] 保存中、错误摘要和必填校验仍可用。
- [x] 前端构建和 `./scripts/verify.sh` 通过。

  输出 `Verification passed`。
